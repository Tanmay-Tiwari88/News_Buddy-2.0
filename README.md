# News_Buddy-2.0

#Introduction
  It is as side pane Electron application. 
  -You can read live news from various categories 
  -Search news
  -Save News for later read
  -create Albums
  -Apply filter on your searches
  

#Launguages Used
-JavaScript

#Technology
-Node js
-Electron 
-Mongodb
-mongoose


#Installion

clone the repo, open folder in terminal
  -install mongodb 
  -run npm install
  -run npm start
